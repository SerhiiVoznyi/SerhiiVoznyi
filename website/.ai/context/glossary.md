# Glossary

Project-specific terms and acronyms. Alphabetized. Add entries as they emerge.

| Term | Definition |
| --- | --- |
| ADR | Architecture Decision Record. Lives under `.ai/memory/decisions/`. |
| Agent | An AI actor operating in this repo, configured by files under `.ai/agents/`. |
| DoD | Definition of Done — see [`.ai/checklists/definition-of-done.md`](../checklists/definition-of-done.md). |
| Skill | A single reusable capability (verb), defined in `.ai/skills/<name>/SKILL.md`. |
| Workflow | A multi-step orchestration composing skills, defined in `.ai/workflows/<name>/workflow.md`. |
