# checklists

Runnable gates used by agents and humans. Each checklist is short, specific, and lives in one place.

## Checklists

- [`pre-commit.md`](pre-commit.md) — quick local checks before committing.
- [`pull-request.md`](pull-request.md) — what every PR must satisfy.
- [`release.md`](release.md) — cutting a release from `main`.
- [`incident.md`](incident.md) — first actions during an incident.
- [`definition-of-done.md`](definition-of-done.md) — what "done" means for any task.

## Adding a checklist

1. Copy [`../templates/checklist.template.md`](../templates/checklist.template.md).
2. Keep it scoped and short. If it balloons, split it.
